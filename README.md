# Crisis Wordlist Generator

Work on Windows, Debian and Kali Linux. 

# Run
<pre style=" border: 1px solid black; padding:10px">
root@kali:~# crisis 
</pre>

# Manual

<pre style=" border: 1px solid black; padding:10px">
root@kali:~# crisis -h
</pre>
